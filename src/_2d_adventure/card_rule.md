每回合最多出 3 张牌。卡牌分为五类：攻击、 治疗、控制、召唤、强化。

**攻击牌**
   - ⚔️：直接削减敌人生命，分为【单体】或【群体】攻击。

**治疗牌**
   - ❤️：生命恢复。芙提雅固有血量 = 【好感度】数值。
   - 🛡️：护甲增加。护甲可临时提高血量，并可跨战斗保留。

**控制牌**
   - 🔇：沉默，让敌人 N 回合内无法发动技能。
   - ❄️：冻结，让敌人 N 回合内无法进行任何行动。
   - 💥：易伤，让敌人 N 回合内受到更多伤害。

**召唤牌** (芙提雅操纵火种发动攻击)
   - 🔥：固有攻击伤害，可【单体】或【群体】攻击，数值低于同等级攻击牌。
   - 🩸：召唤牌附带的【流血】效果，敌人死亡前，每回合都会持续扣血。

**强化牌** (buff 叠加积攒高额爆发)
   - ✨：对芙提雅的增益效果
   - 🔽：对敌人的减益效果
   - 增减益效果会在玩家和敌方单位头顶显示【状态标签】，点击可查看详情。

分析员拥有两个特别技能：
- **神之守护**：使芙提雅生命完全回满，并沉默敌人 1 回合。
- **御驾亲征**：选择一个目标，造成 99999999 点伤害。不能对 BOSS 使用。
